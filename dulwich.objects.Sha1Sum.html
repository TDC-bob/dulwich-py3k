def __init__(self, sha, error_message=None, resolve=False, lazy_errors=False):
Create a SHA-1 sum
ParametersshaOne of (a) a length-20 raw bytes object, (b) a length-40 bytes object consisting of ascii characters only, or (c) a length-40 python string.
error_messageAn error message to return if there's a problem
resolveAutomatically fill in the other representations of this sha1-sum. If False, the other representations get computed on demand.
lazy_errorsIf True, no errors are thrown until someone asks for the value. Otherwise the errors are thrown immediately.
